About

    
This dashboard is served by AgentKit's runtime for any agent or flow via create_app(). The same object you call .run() on in-process is exposed here over HTTP with live token streaming (SSE), durable runs, MCP/A2A, and these introspection endpoints:

    
    
      
  • POST /invoke — run (JSON or text/event-stream)
    • 
      
  • GET /graph — orchestration topology · GET /info — capabilities
    • 
      
  • GET /runs/{id} · POST /runs/{id}/resume — durable runs
    • 
      
  • POST /mcp · GET /.well-known/agent.json · POST /a2a — protocols
    • 
      
  • GET /metrics · GET /healthz
    •
